# SOC Analyst Portfolio

Hi, I'm Ashley.

I'm currently working towards a career in cyber security after spending most of my career in customer-facing and management roles.

Over the last year I've been studying CompTIA Security+, working through Network+ and building practical experience with home labs and platforms like TryHackMe. I started this repository to document what I'm learning and to show the practical work I'm doing along the way.

## What you'll find here

- Home lab projects
- TryHackMe write-ups
- Vulnerability scanning with OpenVAS
- Network scanning with Nmap
- Windows and Linux notes
- SOC investigations
- Security notes and learning

## Tools I'm using

- Kali Linux
- Windows
- VirtualBox
- Nmap
- OpenVAS / Greenbone
- TryHackMe
- Wireshark
- Metasploit

## Why I made this

I'm a big believer that the best way to learn is by actually doing the work.

Rather than just collecting certificates, I wanted somewhere to document the projects I've completed, the problems I've run into and what I've learnt from them. As I continue studying and gaining experience, I'll keep adding new projects here.

Thanks for taking a look.
